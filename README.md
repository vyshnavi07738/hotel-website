# 🏨 Royal Palace Hotel Website

## 📌 Project Description
Royal Palace Hotel Website is a responsive website developed using HTML, CSS, and JavaScript. It provides users with information about the hotel, available rooms, facilities, gallery, and room booking.

## 🚀 Features
- Home Page
- About Us
- Rooms
- Room Prices
- Facilities
- Gallery
- Contact Page
- Room Booking Form
- Responsive Design

## 💻 Technologies Used
- HTML5
- CSS3
- JavaScript
